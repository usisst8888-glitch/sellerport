## 채널별 처리 방식 요약

| 광고 채널 | 추적 링크 | API 연동 | 비고 |
|----------|:--------:|:--------:|------|
| 메타 (인스타/페북) | ✅ | ✅ (광고비) | 브릿지샵 또는 go 링크 |
| 구글/유튜브 | ✅ | ✅ (광고비) | 브릿지샵 또는 go 링크 |
| 틱톡 | ✅ | ✅ (광고비) | 브릿지샵 또는 go 링크 |
| 카카오 | ✅ | ✅ (광고비) | go 링크 |
| 블로그/카페 | ✅ | ❌ | go 링크 |
| 인플루언서 | ✅ | ❌ | go 링크 |
| **네이버 검색광고** | ❌ | ✅ (전환+광고비) | 네이버 자체 추적 |
| **네이버 GFA** | ❌ | ✅ (전환+광고비) | 네이버 자체 추적 |

## 핵심 포인트

1. **5단계로 끝** - 빠른 온보딩
2. **목표 먼저 선택** - 불필요한 옵션 숨김
3. **네이버 광고는 API만** - 추적 링크 불필요
4. **광고 채널 연동은 선택** - 나중에 해도 됨
5. **바로 추적 링크 발급** - 즉시 사용 가능
