| 도메인 | sellerport.app |

## 핵심 가치

```
광고 집행 → 전환 추적 → 주문 수집 → 수수료 계산 → 마진 계산 → 세금 계산 → 실제 순수익
```

### 핵심 데이터 연동 구조

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           셀러포트 데이터 흐름                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  [1] 슬롯 발급 (전환 추적)                                                 │
│      └── UTM 링크 생성 (source, medium, campaign)                        │
│          └── 상품 연결 (product_id)                                      │
│              └── 광고비 입력/연동 (ad_spend)                              │
│                                                                         │
│  [2] 고객 유입                                                           │
│      └── UTM 링크 클릭 → 픽셀샵 → 스마트스토어 리다이렉트                    │
│          └── _fbp 쿠키 저장 (Meta 전환 추적용)                             │
│          └── 클릭 ID 저장 (Google/Naver 추적용)                           │
│                                                                         │
│  [3] 주문 발생                                                           │
│      └── Commerce API로 주문 감지                                        │
│          └── UTM/쿠키로 어떤 슬롯에서 온 주문인지 매칭                       │
│              └── 슬롯별 매출 집계                                         │
│                                                                         │
│  [4] ROAS 계산 + 신호등                                                  │
│      └── ROAS = (슬롯 매출 / 슬롯 광고비) × 100                           │
│          └── 🟢 300%+ / 🟡 150-300% / 🔴 150% 미만                       │
│              └── 빨간불 시 카톡 알림 발송                                  │
│                                                                         │
│  [5] 마진 자동 계산 (주문별)                                               │
│      └── 판매가 (주문에서)                                                │
│      └── 원가 (상품에 등록된 원가)                                         │
│      └── 플랫폼 수수료 (판매 채널별 자동 계산)                               │
│      └── 배송비 (상품에 등록 or 주문에서)                                   │
│      └── 광고비 (슬롯에 연결된 광고비 ÷ 전환 수)                            │
│      └── 순이익 = 판매가 - 원가 - 수수료 - 배송비 - 광고비                   │
│                                                                         │
│  [6] 대시보드 표시                                                        │
│      └── 슬롯별 ROAS + 신호등                                            │
│      └── 상품별 실제 순이익                                               │
│      └── 전체 통계 (매출, 마진율, ROAS)                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 핵심 테이블 관계

```
slots (슬롯/전환추적)
├── id, user_id
├── product_id → products.id
├── utm_source, utm_medium, utm_campaign
├── ad_spend (광고비)
├── clicks (클릭 수)
├── conversions (전환 수)
└── revenue (매출)

products (상품)
├── id, user_id
├── platform_type (naver, coupang, cafe24...)
├── price (판매가)
├── cost (원가)
└── shipping_cost (배송비)

orders (주문)
├── id, user_id
├── slot_id → slots.id (어떤 슬롯에서 온 주문인지)
├── product_id → products.id
├── order_amount (주문금액)
├── platform_fee (플랫폼 수수료)
├── shipping_cost (배송비)
└── profit (순이익 - 자동 계산)

conversions (전환 추적 로그)
├── id, slot_id
├── click_id, fbp_cookie
├── order_id → orders.id
└── converted_at
```

### 광고 효율 신호등 시스템
| 신호 | 조건 | 알림 |
|------|------|------|
| 🟢 초록불 | ROAS 300% 이상 | "이 광고 효율 좋아요! 예산 늘려보세요" |
| 🟡 노란불 | ROAS 150-300% | "보통이에요. 소재나 타겟 점검 필요" |
| 🔴 빨간불 | ROAS 150% 미만 | "효율 낮아요! 중단 또는 수정 권장" |

**킵그로우와의 차별점:**
- 킵그로우: 메타 전환 추적만 (유료 29,900원 + 네이버 수수료 할인)
- 셀러포트: 모든 광고 채널 전환 추적 + 마진/세금 계산 (슬롯당 2,000원)
