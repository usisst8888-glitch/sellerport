# 셀러포트 서비스 안내

## 광고비는 쓰는데, 어떤 광고가 매출로 이어지는지 모르시겠나요?

셀러포트는 **스마트스토어 셀러**를 위한 **광고 성과 분석 서비스**입니다.

인스타그램, 메타 광고를 운영하시는 셀러분들이 **어떤 광고가 실제 구매로 이어졌는지** 정확하게 확인할 수 있도록 도와드립니다.

---

## 이런 고민이 있으신가요?

- "인스타 광고에 돈을 썼는데, 주문이 광고 때문인지 알 수가 없어요"
- "어떤 게시물이 매출에 도움이 되는지 모르겠어요"
- "광고비 대비 얼마나 벌었는지 계산하기 너무 힘들어요"
- "ROAS가 뭔지는 아는데 직접 계산하려니 복잡해요"

**셀러포트가 이 모든 고민을 해결해드립니다.**

---

## 셀러포트로 할 수 있는 것

### 1. 광고별 매출 추적

각 광고 게시물마다 **전용 추적 링크**를 만들어서 사용하세요.

- 인스타그램 프로필 링크
- 스토리 스와이프 링크
- 메타 유료 광고 링크

이 링크를 통해 들어온 고객이 구매하면, **어떤 광고에서 왔는지 자동으로 기록**됩니다.

### 2. 실시간 성과 확인

대시보드에서 한눈에 확인하세요:

| 항목 | 설명 |
|------|------|
| **클릭수** | 광고 링크를 클릭한 횟수 |
| **전환수** | 실제 구매로 이어진 주문 수 |
| **매출액** | 광고를 통해 발생한 총 매출 |
| **ROAS** | 광고 수익률 (100원 썼을 때 얼마 벌었나) |

### 3. ROAS 신호등으로 쉽게 판단

복잡한 숫자 대신 **신호등 색상**으로 광고 효율을 바로 파악하세요.

- 🟢 **초록불**: 잘 되고 있어요! 광고비보다 3배 이상 벌고 있습니다
- 🟡 **노란불**: 괜찮지만 개선하면 더 좋아요
- 🔴 **빨간불**: 이 광고는 점검이 필요해요

### 4. 메타 광고 자동 연동

메타(페이스북/인스타그램) 광고 계정을 연결하면:

- 광고비가 자동으로 불러와집니다
- 캠페인별 성과를 한눈에 비교할 수 있습니다
- A/B 테스트 중이라면 어떤 광고가 더 효과적인지 알려드립니다

### 5. 쇼핑몰 자동 연동

다양한 쇼핑몰 플랫폼을 연결할 수 있습니다:

| 플랫폼 | 연동 방식 |
|-------|---------|
| 네이버 스마트스토어 | API 연동 |
| 카페24 | OAuth 자동 연동 |
| 아임웹 | OAuth 자동 연동 |

**연동하면:**
- 상품 목록을 자동으로 가져옵니다
- 주문이 들어오면 어떤 광고에서 왔는지 자동 매칭됩니다
- 정산 금액까지 확인할 수 있습니다

### 6. 인스타그램 자동 DM

인스타그램 댓글에 특정 키워드가 달리면 자동으로 DM을 발송합니다.

- 댓글 트리거로 자동 DM 발송
- 추적 링크 자동 포함

## 자주 묻는 질문

### Q. 스마트스토어 외에 다른 쇼핑몰도 연동되나요?
**네이버 스마트스토어**, **카페24**, **아임웹**을 지원합니다.

### Q. 인스타그램 개인 계정도 연동할 수 있나요?
인스타그램 **비즈니스 계정** 또는 **크리에이터 계정**이 필요합니다. 개인 계정은 메타 정책상 연동이 불가능합니다.

### Q. 추적 링크를 클릭하면 고객에게 어떻게 보이나요?
추적 링크를 클릭하면 **바로 상품 페이지로 이동**합니다. 고객은 일반 링크와 차이를 느끼지 못합니다.

### Q. ROAS가 뭔가요?
ROAS(Return On Ad Spend)는 **광고 수익률**입니다.
예를 들어 광고비 10만원을 써서 30만원 매출이 나왔다면 ROAS는 300%입니다.
- ROAS 100% = 본전
- ROAS 200% = 2배 수익
- ROAS 300% = 3배 수익

### Q. 무료 체험 후 자동 결제되나요?
아니요. 무료 체험이 끝나면 **자동으로 결제되지 않습니다**. 계속 사용하시려면 직접 결제를 진행해주세요.

### Q. 데이터는 안전한가요?
네, 모든 데이터는 **암호화**되어 저장됩니다. API 키 등 민감한 정보는 안전하게 보호됩니다.
