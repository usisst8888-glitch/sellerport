## 2. 문제 공감 섹션

**헤드라인:** "이런 경험 있으시죠?"

- 인스타 열심히 올리는데 뭐가 효과 있는지 모름
- 광고비는 나가는데 매출이랑 연결이 안 됨
- 인플루언서한테 협찬 줬는데 효과가 있었는지 모름

## 3. 타겟별 케이스 섹션

### 케이스 A: 인스타 운영하는 초보 셀러

**문제:** "주문 들어왔는데 어떤 게시물 보고 온 건지 모름"

**해결:** 
- 릴스 A → 주문 12건
- 착샷 게시물 → 주문 2건
- 스토리 → 주문 0건

**결론:** "아, 릴스가 진짜 효과 있네. 릴스 더 만들어야겠다"

---

### 케이스 B: 광고비 쓰는 셀러

**문제:** "여기저기 광고하는데 뭐가 효과인지 모름"

**해결:**
- 네이버 광고 → ROAS 320%
- 카카오 광고 → ROAS 85%

**결론:** "네이버에 집중하자!"

---

### 케이스 C: 인플루언서 협찬하는 셀러

**문제:** "협찬비 50만원 줬는데 효과 있었나?"

**해결:**
- 인플루언서 A → 50만원 투자, 매출 12만원
- 인플루언서 B → 30만원 투자, 매출 89만원

**결론:** "B한테 더 줘야겠네"

## 4. 신호등 시스템 섹션

**강조 포인트:** 빨간불 뜨면 **카톡으로 즉시 알림** → 광고비 낭비 방지

## 5. 마진/세금 자동 계산 섹션

**헤드라인:** "진짜 남는 돈, 자동으로 계산해드려요"

**기능 설명:**
- 플랫폼 수수료 자동 반영 (네이버 정산 API 연동)
- 원가, 배송비, 광고비 → 순이익 자동 계산
- 부가세, 종소세 예상 금액까지

**공감 카피:** "매출 100만원인데 실제로 얼마 남았지?" → 바로 확인 가능

## 6. 카톡 알림 섹션

**헤드라인:** "중요한 건 카톡으로 바로 알려드려요"

- **주문 알림:** 유입경로 + 상품 + 마진 포함
- **빨간불 경고:** 효율 낮은 광고 즉시 알림
- **일일 요약:** 매일 밤 당일 성과 리포트
- **AI 추천:** "이 광고 타겟 바꿔보세요" 제안
