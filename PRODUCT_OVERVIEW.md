# 셀러포트 (SellerPort)

> 스마트스토어 셀러를 위한 올인원 광고 성과 분석 플랫폼

---

## 서비스 개요

셀러포트는 네이버 스마트스토어, Cafe24 등 이커머스 셀러들이 **인스타그램, Meta 광고**의 성과를 정확하게 측정하고 분석할 수 있도록 돕는 SaaS 플랫폼입니다.

기존에 광고비는 쓰지만 실제로 어떤 광고가 매출로 이어졌는지 알기 어려웠던 셀러들의 pain point를 해결합니다.

---

## 핵심 기능

### 1. 추적 링크 생성 및 관리

광고별로 고유한 추적 링크(UTM 파라미터 포함)를 생성하여 유입 경로를 정확하게 파악합니다.

- **자동 UTM 생성**: 상품과 광고 채널을 선택하면 자동으로 추적 링크 생성
- **간편한 링크 복사**: 원클릭으로 링크 복사하여 광고에 바로 적용
- **브릿지 페이지**: 클릭 추적을 위한 중간 페이지 자동 생성

**사용 예시:**
```
원본 상품 URL: https://smartstore.naver.com/mystore/products/12345
→ 추적 링크: https://go.sellerport.io/abc123
```

### 2. 광고 성과 대시보드

실시간으로 광고 성과를 한눈에 확인할 수 있습니다.

| 지표 | 설명 |
|------|------|
| 클릭수 | 추적 링크를 통한 유입 수 |
| 전환수 | 실제 구매로 이어진 주문 수 |
| 매출액 | 전환된 주문의 총 매출 |
| 광고비 | Meta 광고 API 연동을 통한 실제 지출액 |
| ROAS | 광고 수익률 (매출/광고비 × 100) |

**ROAS 신호등 시스템:**
- 🟢 **좋음** (300% 이상): 광고비 대비 3배 이상 매출
- 🟡 **보통** (150~300%): 수익은 나지만 개선 필요
- 🔴 **주의** (150% 미만): 광고 효율 점검 필요

### 3. 쇼핑몰 연동

다양한 이커머스 플랫폼과 연동하여 상품/주문/정산 데이터를 자동 동기화합니다.

**지원 플랫폼:**
- ✅ 네이버 스마트스토어 (Commerce API)
- ✅ Cafe24 (OAuth 연동)
- 🔜 아임웹 (예정)

**동기화 데이터:**
- 상품 목록 (이름, 가격, 이미지, 재고)
- 주문 내역 (주문번호, 상품, 결제금액, 상태)
- 정산 정보 (수수료, 정산예정금액)

### 4. 광고 채널 연동

Meta(Facebook/Instagram) 광고 계정을 연동하여 광고비 데이터를 자동으로 가져옵니다.

**연동 채널:**
- ✅ Meta 광고 (Facebook/Instagram Ads)
- ✅ 인스타그램 비즈니스 계정

**가져오는 데이터:**
- 캠페인별 광고비 지출
- 노출수, 클릭수
- 크리에이티브 썸네일
- A/B 테스트 성과 비교

### 5. Meta A/B 테스트 분석

동일 캠페인 내 여러 크리에이티브의 성과를 비교 분석합니다.

- 크리에이티브별 ROAS, CTR, CPC 비교
- 자동 승자 판정 및 인사이트 제공
- 저성과 광고 중단 권장

### 6. 인스타그램 자동 DM (예정)

인스타그램 댓글에 특정 키워드가 달리면 자동으로 DM을 발송하는 기능입니다.

- 키워드 트리거 설정
- 자동 응답 메시지 커스터마이징
- 추적 링크 자동 포함

## 사용자 플로우

```
1. 회원가입/로그인
      ↓
2. 쇼핑몰 연동 (네이버 스마트스토어)
      ↓
3. 상품 동기화
      ↓
4. 광고 채널 연동 (Meta/Instagram)
      ↓
5. 추적 링크 생성
      ↓
6. 광고에 추적 링크 적용
      ↓
7. 광고 성과 확인 (대시보드)
```

## 기술 스택

### Frontend
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State**: React Hooks

### Backend
- **API**: Next.js API Routes
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth

### 외부 연동
- **네이버 커머스 API**: 상품/주문/정산 조회
- **Cafe24 API**: OAuth 연동 및 데이터 동기화
- **Meta Marketing API**: 광고 성과 데이터
- **Instagram Graph API**: 비즈니스 계정 연동

### 결제
- **토스페이먼츠**: 정기결제 (빌링키)
